# Status Vocabulary

Requirement:

```text
Discovered -> Classified -> Accepted -> Allocated -> In Progress -> Claimed Satisfied -> Verified -> Approved
```

Work package:

```text
Draft -> Planned -> Constraint Blocked -> Ready -> Released -> In Execution -> Evidence Submitted -> Reviewed -> Accepted -> Closed
```

Evidence:

```text
Captured -> Linked -> Validated -> Reviewed -> Accepted
```

Value:

```text
Unbudgeted -> Budgeted -> Committed -> Planned -> Earned -> Certified -> Billed -> Approved for Payment -> Paid -> Collected
```
