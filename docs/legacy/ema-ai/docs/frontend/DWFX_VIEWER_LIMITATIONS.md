# DWFx Viewer Limitations (MVP)

DWFx files are indexed as `dwfx_export` evidence candidates.

Current MVP:

- Metadata indexing and listing supported.
- Full browser rendering of DWFx is not implemented.
- UI must present DWFx as indexed 3D export candidate metadata, not as rendered 3D geometry.

Future production path may use APS/Forge or another approved viewer stack.
