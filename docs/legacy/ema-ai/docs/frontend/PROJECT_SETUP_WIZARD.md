# Project Setup Wizard

The Project Setup page implements the local-first setup flow:

1. Create or bind client.
2. Create project shell.
3. Create model shell (optional).
4. Configure landing root + project folder.
5. Bootstrap from existing landing folder when needed.
6. Generate Revit binding JSON for manual paste into Revit settings.

Notes:
- This workflow does not auto-submit ingestion.
- Processing/Sync remains operator-controlled.
- Indexed files are evidence candidates by default.
