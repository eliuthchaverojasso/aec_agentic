# Landing Zone Architecture

## Purpose
Standard folder contract for project artifacts before backend ingest.

## Standard Folders
- Revit Exports
- Drawings
- Specifications
- Owner Requirements
- 3D Exports
- Viewpoints
- Timeline
- Supporting

## Rules
- Path-safe, project-scoped processing only.
- Indexed files default to evidence candidates.
- Manifest-driven scan/rebuild/ingest.
