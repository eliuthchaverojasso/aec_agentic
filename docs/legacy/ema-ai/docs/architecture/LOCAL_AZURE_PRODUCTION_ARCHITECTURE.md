# Local / Azure / Production Architecture

## Local (Implemented)
- Dockerized backend + PostgreSQL.
- Vite frontend.
- Local landing folders and operator-controlled ingest.

## Azure Pilot (Planned)
- Managed storage/data lake and managed PostgreSQL.
- Managed API/frontend hosting.
- Environment mapping and diagnostics parity.

## Production (Planned)
- Auth/RBAC, governance, backup/DR, monitoring, and audit posture hardening.

This document is architecture direction, not proof of deployed infrastructure.
