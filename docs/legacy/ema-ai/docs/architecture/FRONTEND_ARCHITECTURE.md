# Frontend Architecture

## Purpose
React/Vite TypeScript dashboard for setup, processing, readiness views, and debug workflows.

## Components
- `App.tsx` shell and navigation.
- `api/client.ts` typed API layer.
- Project Setup, Processing / Sync, Debug / Logs, System Health pages.
- Selected project persistence via localStorage.

## Status
- Implemented Local: core DD/CD flow and diagnostics views.
- Partial: some pages still include demo placeholders.
- Planned: production auth UX and enterprise theming hardening.
