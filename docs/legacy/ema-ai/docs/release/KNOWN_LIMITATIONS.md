# Known Limitations

- Local MVP, not production-ready.
- No production auth/RBAC enforcement.
- Backend ingest is operator-controlled.
- Evidence candidates are not official evidence by default.
- No official compliance automation from AI/semantic retrieval.
- DWFx browser rendering is not implemented in MVP.
- Windows host landing path vs `/app/landing` container mapping can require manual alignment.
- Debug logs are diagnostic, not audit-compliant telemetry.
