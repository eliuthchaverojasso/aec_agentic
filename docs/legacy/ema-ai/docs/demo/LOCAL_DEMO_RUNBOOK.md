# Local Demo Runbook (Project Setup Workflow)

1. Open **Project Setup** page.
2. Create project/client/model and configure landing root/folder.
3. Or use **Discover Landing Projects** then **Bootstrap From Folder**.
4. Confirm generated Revit binding JSON and copy into Revit add-in settings.
5. Open **Processing / Sync**.
6. Run: Scan Landing -> Rebuild Manifest -> Dry Run Ingest -> Run Ingest.
7. Open dashboard pages (Overview, Documents, Requirements, Trade Readiness) with selected project.

Notes:
- Ingest is operator-controlled.
- Indexed files are evidence candidates by default.
