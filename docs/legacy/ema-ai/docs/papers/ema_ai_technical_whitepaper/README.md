# EMA-AI Technical Whitepaper

## Build

From this directory run:

```powershell
latexmk -pdf -interaction=nonstopmode -halt-on-error main.tex
```

## Notes

- Source files are intended to be committed.
- Generated artifacts such as `.aux`, `.log`, `.out`, `.toc`, `.fls`, `.fdb_latexmk`, and `.synctex.gz` should not be committed.
- The paper is written for a local MVP context and does not claim production readiness or official compliance.
