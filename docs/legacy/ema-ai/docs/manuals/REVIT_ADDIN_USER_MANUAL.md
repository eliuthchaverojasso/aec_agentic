# Revit Add-in User Manual

## Start
1. Install/update add-in (`docs/revit/ADDIN_INSTALLATION.md`).
2. Open Revit and confirm `EMA AI` tab.

## Workflow
1. Connect Project / select binding.
2. Configure Landing Settings.
3. Validate Landing.
4. Export all or discipline-specific JSON.
5. Confirm metadata sidecar creation.
6. Open web dashboard and run Processing / Sync ingest.

## Boundary
Revit export writes landing artifacts and metadata. Official ingestion and readiness updates are operator-controlled from web app Processing / Sync.
