# Revit Runtime Smoke Checklist

## Purpose

Checks the EMA-AI add-in in a live Revit host when that host is available.

## Checklist

- EMA AI tab appears
- Project Setup commands appear
- Export commands appear
- Document Intake commands appear
- Sync / Backend commands appear
- Open Dashboard command works
- Landing folder open commands work

## Caveat

Runtime smoke is only valid if actually performed inside Revit.
