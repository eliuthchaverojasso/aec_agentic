# Doctrine

ORGANISM is a local, persistent, evolvable cognitive system. It survives restarts because missions, decisions, events, artifacts, memories, and approved methods are durable.

Core principles:

- Continuity is stored, not assumed.
- Models are replaceable workers, not identity.
- Git holds doctrine, policies, skills, and methodologies.
- PostgreSQL holds operational mission state.
- Event logs preserve history.
- Artifacts preserve observable reality.
- pgvector and Apache AGE are derived, rebuildable indexes.
- Human approval gates changes to doctrine, governance, destructive actions, publishing, deployment, and self-modification.
