# Security

This repo must never contain customer data, secrets, real project exports, Revit models, database dumps, or credentials.

Agents, connectors, and apps must preserve these boundaries:

- Source-system payloads are immutable inputs.
- AI outputs are advisory until reviewed.
- Authority-bearing decisions require explicit actor identity.
- Economic recognition requires deterministic policy and audit.
- External credentials belong in secret stores or local ignored files, never source.

Report security issues privately to the repository owner.
